# Baseline Git - V1 oficial local

Data: 2026-05-28

## Estado

Baseline local da V1 preparado e evoluido ate a linha oficial de release IEDC.

O primeiro release candidate (`v1.0.0-local-rc1`) foi criado apos a auditoria de RBAC, QA por perfil, Docker e backup/restauracao local.

O release candidate atual incorporou a decisao de bootstrap institucional: em banco novo, nasce apenas o usuario `suporte`.

A tag tecnica final planejada para o repositorio oficial e `v1.0.0-iedc`. O titulo humano da release pode ser `v1.0.0 iedc`.

## Validacoes executadas

- Backend: `npm test -- --runInBand` aprovado.
- Backend: `npm run build` aprovado.
- Frontend: `npm run build` aprovado.
- Docker: `docker compose up -d --build frontend` aprovado.
- Healthcheck: `/api/health` aprovado.
- QA por perfil: 11 perfis aprovados.
- Usuarios temporarios `qa-*`: removidos apos validacao.
- Backup/restauracao: scripts validados em Docker local com backup temporario.
- Primeiro boot: banco novo cria apenas o usuario `suporte`, com senha temporaria de `IEDC_DEFAULT_PASSWORD`.

## Alertas conhecidos

- Frontend ainda pode emitir alerta de chunks grandes no build. Nao bloqueia a V1 local; deve virar tarefa de refinamento de bundle depois do baseline.
- Auditoria de dependencias backend foi endurecida na sprint de seguranca V1 e deve continuar como gate de release.
- Restauracao de backup foi validada em Docker local; repetir no servidor institucional antes de tratar a tag como release implantavel.
